1. This is a front end program file using stremlit.
2. No changes are required in a program file.
3. use pip command to install requirements.txt file.
	pip install requirements.txt
3. run the main.py file and this will provide you with the url needed for stremlit.
5. You will get output as: to view this Streamlit app on a browser, run it with the following
  command:

    streamlit run ***/Users/sachinsingh/PycharmProjects/depression_detc/main.py*** #this will give you path of your file.

6. Open python terminal and copy paste the path provided above. This will provide you with localhost URL where you can access the front model. It must look like this:
     You can now view your Streamlit app in your browser.

      Local URL: http://localhost:8501
      Network URL: http://10.0.0.164:8501

7. In front-end you can select lSTM or CNN model from left list, type a tweet and check the state of tweet in output section.
